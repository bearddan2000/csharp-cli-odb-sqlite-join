# csharp-cli-odb-sqlite-join

## Description
A c# running local sqlite database.
The query is a JOIN of two tables.

## Tech stack
- bash
- c#
  - Framework 4.7
- sqlite3
- mono 6.8

## Docker stack
- docker-cli
- ubuntu:20.04

## Requirements
Docker desktop must be installed and the application
being called must be linux compatiple.

## To run
`sudo ./install.sh -u`

## To stop (optional)
`sudo ./install.sh -d`

## To see help
`sudo ./install.sh -h`
